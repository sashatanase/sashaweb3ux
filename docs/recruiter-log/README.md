# Recruiter log

Daily verdicts from the [Recruiter Agent](../../.github/recruiter-agent/).

Each run appends one entry to a monthly file (`YYYY-MM.md`) recording the date,
the decision (**CONTACT** or **IMPROVE**), a short rationale, and a link to the
issue or PR it produced. This is the running record of how the portfolio is
trending in a recruiter's eyes over time.
